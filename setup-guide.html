🛠️ Setup Steps (15 minutes)

1 Create GitHub account

Go to github.com/signup, sign up with your email.

2 Subscribe to Claude Pro

Go to claude.ai, create account, upgrade to Pro ($20/month).

3 Install Claude Code (on your computer)

Open Terminal (Mac) or Command Prompt (Windows) and type:

npm install -g @anthropic-ai/claude-code

Then start it by typing:

claude

Log in with your Claude account. Done!

⚠️ Note: You need Node.js installed first. Download from nodejs.org (choose LTS version). This is a one-time setup.

💬 How to Use (Just Talk to It!)

Once Claude Code is running, just type what you want in plain English:

"Make me a landing page for my Mandarin tutoring class"
→ Creates a complete website with your info
"Write a Facebook post to recruit new students"
→ Writes engaging copy with hashtags
"Create a 5-page presentation about my class"
→ Makes interactive slides you can present
"Build an interactive picture book to teach numbers in Chinese"
→ Creates a web app where kids tap to hear pronunciation
"Make worksheets for learning basic Chinese characters"

"Deploy my website to GitHub Pages"
→ Publishes your site with a shareable link
